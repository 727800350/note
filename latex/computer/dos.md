**进入D盘, `cd D:\` 不管用**  
```
D: ENTER
```

Combine 2 text files into one:
```
type file1 file2 > file3
```
DOS:
```
copy file1+file2 file3
```
Create a text file from the keyboard: copy con: file1 {press Ctrl-Z to end}

Write a text file to the screen: 
```
type file
```
