## Example

    {
	    "firstName": "John",
	    "lastName": "Smith",
	    "isAlive": true,
	    "age": 25,
	    "height_cm": 167.64,
	    "address": {
	    "streetAddress": "21 2nd Street",
	    "city": "New York",
	    "state": "NY",
	    "postalCode": "10021-3100"
    	},
	    "phoneNumbers": [
		    { "type": "home", "number": "212 555-1234" },
		    { "type": "fax",  "number": "646 555-4567" }
	    ]
    }

## Format

1. 并列的数据之间用逗号(",")分隔. 
2. 映射用冒号(":")表示. 
3. 并列数据的集合(数组)用方括号("[]")表示. 
4. 映射的集合(对象)用大括号("{}")表示.

comments in JSON are not allowed.
也就是说JSON中全是data
